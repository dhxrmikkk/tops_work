What is Form ?

Ans : form is used to get input from user

Form has two methods in html

  1. GET
  2. POST

1) GET Method

  1. Get is By Default method
  2. If we pass data with GET method quary string will be occured
  3. Example of quary string : ?email=dharmik%40gmail.com&password=12345&btn-login=Submit
  4. Less secure than post method

1) POST Method

  1. If we pass data with POST method quary string will be not occured
  2. More secure than GET method
