what is void elements ?

Ans : In Html there are some void elements which doesn't have ending tags

They are also call as null elements or self closing tags

Examples of self closing tags are listed below

  1. < area >
  2. < base >
  3. < br >
  4. < hr >
  5. < img >
  6. < link >
  7. < meta >
  8. < track >
  9. < param >
  10. < source >
